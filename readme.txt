Crypto Net - Cryptocurrency Market Cap, Live Prices, Chart, Ticker & Calculator


With Crypto Net PHP Script, you can create a fully automated crypto real time market data website within minutes which displays the market overview, live prices, market cap, available supply, historical coin data in chart, 24 hours volume, trading pairs, cryptocurrency calculator and price changes for more than 9000+ coins. Along with this you can show exchanges list of 300+ exchanges along with exchange single page which shows exchange market data, trading pairs, volume chart and much more. Also, you can display the Top 25 cryptocurrency ticker in the footer.

http://cryptonet.techpellet.net/documentation

To view documentation offline, Please Click on index.html file in the documentation folder.
